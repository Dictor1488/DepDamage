# Battle vehicle markers AS3 source

This directory is reserved for AS3 source of `battleVehicleMarkersApp.swf`.

The source must be built into:

```text
as3/bin/battleVehicleMarkersApp.swf
```

Then `build.py` packages `as3/bin` into the `.wotmod` as:

```text
res/gui/flash/battleVehicleMarkersApp.swf
```

Do not commit old Gameface files, old SWF binaries, or the previous `FlyingDamageApp` overlay renderer here.
