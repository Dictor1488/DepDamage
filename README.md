# Mastery — конфігурація

Мод сам створює файл конфігу при першому запуску:

```
<game>/mods/configs/mastery/mastery.json
```

## Ключі

### `mode`
Що показувати:
- `both` — панель Masters + бойова мітка (за замовчуванням)
- `mastery` (або `masters`) — лише панель Masters, мітка вимкнена
- `marks` (або `mark`) — лише бойова мітка, панель Masters вимкнена

### `badgeStyle`
Вигляд бойової мітки (в бою та в ангарі):
- `classic` — оригінальний вигляд: зірки і значення по центру (за замовчуванням)
- `compact` — новий вигляд: зірки + вертикальні смужки зліва, велике значення
  справа, діагональна стрілка тренду, дрібний ±%, квадратний прогрес-бар,
  рядок «Сум. урон  поточний / поріг»

Синоніми: `old`/`original` = `classic`; `new`/`left` = `compact`.

## Приклад

```json
{
    "mode": "both",
    "badgeStyle": "compact"
}
```

Після зміни файлу перезапустіть гру (або перезайдіть в ангар).
